# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be kind and constructive.
- Assume good intent.
- Keep discussions focused on the package and its users.
- Respect privacy and never post real credentials or private customer/payment data.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without consent.
- Posting real tokens, API keys, card numbers, IBANs, national codes, mobile numbers, or private callback URLs.

## Enforcement

Maintainers may edit, hide, or remove comments and may block participants who do not follow this code of conduct.
