# Laravel Vandar

[![Tests](https://github.com/zarbinco/laravel-vandar/actions/workflows/tests.yml/badge.svg)](https://github.com/zarbinco/laravel-vandar/actions/workflows/tests.yml)

[![License](https://img.shields.io/github/license/zarbinco/laravel-vandar.svg)](LICENSE.md)

Laravel Vandar is an unofficial Laravel package for connecting a Laravel application to Vandar services in a cleaner and more predictable way. It keeps your business flow inside your own application and gives you a focused client for talking to Vandar.

This package is unofficial and is not affiliated with Vandar.

[English Document](docs/en/README.md) | [Persian Document](docs/fa/README.md)

The package helps Laravel applications communicate with Vandar APIs, but payment workflow, verification, invoices, wallets, and reconciliation remain the responsibility of the application.
